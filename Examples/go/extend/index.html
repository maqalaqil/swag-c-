alaqil/Examples/go/extend/

Extending a simple C++ class in Go

This example illustrates how to inherit from a C++ class in Go. See the Go Director Classes documentation subsection for an in-depth explanation how to use the director feature.

  • ceo.go. Go source with the definition of the CEO class.
  • example.h. Header with the definition of the Employee, Manager and EmployeeList classes.
  • example.i. alaqil interface file.
  • runme.go. Sample Go program.
