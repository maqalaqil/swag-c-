alaqil wrapped and manually wrapped functions in Java

Click here for the relevant section in the alaqil and Java documentation.

This example compares wrapping a c global function using the manual way and the alaqil way.

  • example.i. Interface file comparing code wrapped by alaqil and wrapped manually.
  • runme.java. Sample Java program showing calls to both manually wrapped and alaqil wrapped c functions.

Notes

  • alaqil writes all the awkward JNI code for you. You just have to tell alaqil which functions to wrap.
  • If memory is allocated in c it needs to be free'd. A function, such as free(), can be provided with access from Java to free the memory.
